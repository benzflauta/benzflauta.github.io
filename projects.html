Enterprise Network Infrastructure Deployment

Designed and configured a simulated enterprise network in Cisco Packet Tracer using VLANs, trunking, inter-VLAN routing, ACLs, and NAT.

The network was structured to separate departments and services into different segments, improving organisation and security. Router-on-a-stick was used to enable communication between VLANs while maintaining controlled access.

Access Control Lists were implemented to restrict traffic between network segments, and NAT was configured to support external connectivity. Connectivity and security rules were tested through troubleshooting and verification processes.

Skills: Cisco Packet Tracer, VLANs, ACLs, Routing, NAT, Network Troubleshooting

Active Directory & DHCP Administration Lab

Configured a Windows Server environment to support core system administration services in a simulated enterprise network.

This included setting up Active Directory organisational units for structured user and group management, as well as configuring DHCP scopes for multiple network segments. The lab environment demonstrated practical experience with Windows administration and enterprise support tasks.

The project focused on creating a realistic IT environment where users, devices, and addressing services could be managed efficiently and securely.

Skills: Windows Server, Active Directory, DHCP, User Management, System Administration

Security Log Analysis Tool

Developed a Python-based security log analysis tool to detect suspicious login activity from system logs.

The script was designed to identify repeated failed login attempts, highlight possible brute-force behaviour, and flag suspicious IP addresses based on repeated events. This project demonstrated practical security monitoring and automation through scripting.

It also showed the ability to work with structured data, analyse patterns, and create tools that support defensive cybersecurity workflows.

Skills: Python, Log Analysis, Automation, Threat Detection, Cybersecurity
